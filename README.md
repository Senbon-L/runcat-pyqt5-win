# runcat-pyqt5-win

[RunCat](http://kyomesuke.com/runcat/index.html) – mac应用，用奔跑的猫来显示当前系统资源占用情况。

找了一圈只有mac版，于是用pyqt5撸一个，在Windows任务栏（通知区域）养猫。


## Requirements
- psutil
- pyqt5


## Usage

- 直接clone， 改 *.pyw 运行

- 或者下载打包后的 *.exe， 但是有30多M。。。 [下载](https://github.com/shenbo/runcat-pyqt5-win/releases)
